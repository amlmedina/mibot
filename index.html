Para desarrolladores

API REST lista para usar

Integra MiBot en minutos desde cualquier lenguaje.

  • ✓ Autenticación por header x-api-key
  • ✓ Respuestas JSON estándar
  • ✓ Envío inmediato y programado
  • ✓ Documentación completa en el panel
// Apps Script — enviar WA desde Sheets
const KEY = PropertiesService
  .getScriptProperties()
  .getProperty("MIBOT_KEY")

function enviarWhatsApp(numero, mensaje) {
  return JSON.parse(
    UrlFetchApp.fetch(
      "https://api.mibot.mx/api/send", {
        method: "post",
        headers: { "x-api-key": KEY },
        payload: JSON.stringify({
          to: numero, message: mensaje
        })
    }).getContentText()
  )
}
# Python — programar mensajes de WhatsApp
import requests, os

URL = "https://api.mibot.mx"
KEY = os.environ["MIBOT_KEY"]
H = {"x-api-key": KEY}

def programar(numero, mensaje, fecha):
    # fecha: "2026-05-01T09:00" hora CDMX
    return requests.post(
        f"{URL}/api/schedule",
        headers=H,
        json={
          "to": numero,
          "message": mensaje,
          "sendAt": fecha
        }
    ).json()

FAQ

Preguntas frecuentes

¿Necesito la API oficial de WhatsApp Business?+
No. MiBot funciona con tu número de WhatsApp normal. Solo escaneas un código QR desde el panel y en menos de 30 segundos estás conectado.
¿Puedo integrar MiBot con Google Sheets?+
Sí. El panel incluye código listo para copiar y pegar en Google Apps Script. También puedes importar mensajes masivamente pegando directamente desde tu hoja de cálculo.
¿Funciona en toda Latinoamérica?+
Sí. MiBot funciona con cualquier número de WhatsApp de cualquier país: México, Colombia, Argentina, Chile, Perú, Venezuela y más.
¿Qué pasa si mi cuenta de WhatsApp es baneada?+
El uso de automatizaciones puede violar los Términos de Servicio de WhatsApp. Recomendamos usarlo con moderación y solo enviar mensajes a personas que dieron su consentimiento. MiBot no se hace responsable por suspensiones.
¿Los mensajes se envían aunque cierre el panel?+
Sí. El servidor funciona 24/7 en la nube. Una vez que programas un mensaje, se envía automáticamente a la hora indicada sin que tengas que hacer nada más.
¿Puedo cancelar en cualquier momento?+
Sí. Cancela cuando quieras desde el portal de Stripe. Sin contratos ni permanencias. Mantienes acceso hasta el final del período pagado.
